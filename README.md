Project Details
1. Name of your project
KeenKeeper – A Personal Friendship Management System.

2. A short description
KeenKeeper is a modern web application designed to help users maintain and nurture their meaningful relationships. It provides a visual dashboard to track interactions with friends, monitor communication status and log history. The goal is to ensure no important connection is ever neglected in the bustle of daily life.

3. Technologies I used
Framework: React.js (Vite)
Styling: Tailwind CSS (for a clean and responsive UI)
Routing: React Router DOM
Icons: Lucide-React
Charts: Recharts (for interaction analytics)
Notifications: React Hot Toast
State Management: React Context API (to manage timeline history)
Data Handling: JSON-based data fetching

4. Three Key Features of my project
Smart Relationship Tracking: Categorizes friends with visual status badges (Overdue, Almost Due, On-track) based on the last time you contacted them, helping you prioritize who needs attention.
Automated Interaction Timeline: A dedicated history page that logs every call, text, or video interaction automatically. It includes advanced filtering options to quickly view specific types of communication.
Insightful Friendship Analytics: An interactive stats page featuring a Pie Chart that visualizes your communication habits, showing the distribution of calls, texts, and video chats.
